# Objetivo da Disciplina

A Visão Computacional é uma das áreas mais importantes da Inteligência Artificial moderna. Seu objetivo é permitir que sistemas computacionais **interpretem e compreendam imagens e vídeos**, automatizando tarefas que antes dependiam exclusivamente da percepção humana.

No contexto empresarial, essas tecnologias estão sendo utilizadas para:

- Automação de processos industriais
- Monitoramento de segurança do trabalhador
- Análise de comportamento de consumidores
- Sistemas inteligentes de vigilância
- Análise de imagens médicas
- Veículos autônomos e robótica
- Busca e organização de conteúdo visual

Durante a disciplina exploramos **conceitos fundamentais, tecnologias utilizadas no mercado e exemplos práticos de aplicação**.

# Como Utilizar Este Material

Os notebooks podem ser executados de duas formas.

## Executando no Google Colab

1. Acesse o notebook desejado  
2. Abra no **Google Colab**  
3. Execute as células diretamente no navegador  

## Executando localmente

Instale as dependências básicas:

```bash
pip install tensorflow opencv-python numpy matplotlib jupyter
```

Depois execute:

```bash
jupyter notebook
```
