# Visão Computacional para Negócios

Material da disciplina **Visão Computacional para Negócios** do **MBA em Gestão de Negócios da USP/ESALQ**.

Este repositório reúne **notebooks, exemplos práticos e materiais complementares** utilizados em aula, com o objetivo de demonstrar como técnicas modernas de **Inteligência Artificial e Computer Vision** estão sendo aplicadas em diferentes contextos empresariais.

A disciplina apresenta os conceitos de forma **acessível para profissionais de negócios**, conectando fundamentos da tecnologia com **casos reais de uso em empresas e indústrias**.

## Objetivo da Disciplina

A Visão Computacional é uma das áreas mais importantes da Inteligência Artificial moderna. Seu objetivo é permitir que sistemas computacionais **interpretem e compreendam imagens e vídeos**, automatizando tarefas que antes dependiam exclusivamente da percepção humana.

No contexto empresarial, essas tecnologias têm sido aplicadas para:

- Automação de processos industriais
- Monitoramento de segurança do trabalho
- Análise de comportamento de consumidores
- Sistemas inteligentes de vigilância
- Análise de imagens médicas
- Veículos autônomos e robótica
- Busca e organização de conteúdo visual

Esta disciplina apresenta **conceitos fundamentais e exemplos práticos**, com foco na compreensão do impacto dessas tecnologias nos negócios.

## Como Utilizar Este Material

Os notebooks podem ser executados de duas formas:

### Executando no Google Colab

1. Acesse o notebook desejado  
2. Abra no **Google Colab**  
3. Execute as células diretamente no navegador  

### Executando localmente

Instale as dependências básicas:

```bash
pip install tensorflow opencv-python numpy matplotlib jupyter
```

Depois abra os notebooks com:

```bash
jupyter notebook
```
